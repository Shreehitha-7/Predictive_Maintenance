# Predictive Maintenance using Machine Learning
## Overview
This project detects anomalies in boiler temperature and pressure data to predict equipment failure.
## Features
- Detects abnormal sensor behavior
- Uses machine learning for prediction
- Visualizes trends using graphs
## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
## Dataset
Simulated industrial boiler dataset with temperature and pressure readings.
## How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the notebook
## Output
Shows temperature & pressure trends and detects anomalies
